      --Cadastro de Navegantes.

  -Este código em Python cria uma aplicação simples 
 de desktop usando Tkinter para gerenciar um cadastro 
 de navegantes.

   Funcionalidades:

 -Adicionar novos navegantes: Permite inserir dados como 
 nome, telefone, CPF, tipo de embarcação, tipo de pesca e 
 local de pesca.
 -Visualizar lista de navegantes: Exibe uma lista de todos
 os navegantes cadastrados, ordenados por nome.
 -Buscar navegantes: Permite buscar navegantes por nome.
 -Editar dados de navegantes: Permite editar as informações 
 de um navegador selecionado na lista.
 -Excluir navegantes: Permite remover um navegador do cadastro.

   Pré-requisitos:
 1.Instalar um executador de códigos. Ex:.(Visual Studio Code)
 2.Python instalado.
 3.Bibliotecas tkinter, ttk, messagebox e sqlite3 instaladas.

   Instalação:
 1.Crie um novo arquivo chamado cadastro_navegantes.py.
 2.Cole o código fornecido neste arquivo.
 3.Execute o código: python cadastro_navegantes.py

   Uso:
 1.A aplicação abrirá uma janela com duas áreas principais:
	-Área de entrada de dados: Contém campos para inserir as 
	informações do navegador e botões para Limpar, Buscar, 
	Novo, Alterar e Apagar.
	-Lista de navegantes: Exibe uma lista de todos os navegantes 
	cadastrados.
 2.Para adicionar um novo navegador, preencha os campos e clique no botão "Novo".
 3.Para editar os dados de um navegador, selecione-o na lista (clicando duas vezes) 
   e altere as informações desejadas. Em seguida, clique no botão "Alterar".
 4.Para excluir um navegador, selecione-o na lista e clique no botão "Apagar".
 5.Para buscar navegantes por nome, digite o nome no campo de busca e clique no 
   botão "Buscar".
